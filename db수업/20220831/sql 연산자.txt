비교 연산자 

where절에서 문자, 숫자, 날짜의 크기나 순서를 비교함

같을때 :		=
같지 않을때 :	!=, <>
클때: 		>
크거나 같을때: 	>=
작을때:		<
작거나 같을때: 	<=


논리 연산자

where절에서 여러 개의 조건을 결합할 경우

x and y : x,y가 참일때 참을 반환
x or y : x나 y가 참일 때 참을 반환
not x : x가 거짓일때 참을 반환

데이터 출력 순서 

asc : 오름차순으로, 기본값(생략이 가능)
desc : 내림차순, 생략불가능!!
